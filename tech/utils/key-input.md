# Key Input 

## Unicode + Diacritics

- Dead Keys: https://en.wikipedia.org/wiki/Dead_key

### Windows

- `Alt` + Code
- `Intl US` Layout: Use `Alt Gr` + key for common keys
- Emojis: `Win + .` or `Win + ;`

### Linux

- `Ctrl+Shift+u`, Code
- `Ctrl+Shift` + (`u, Hex code`)
- `Intl with Alt Gr Dead Keys` Layout: Use `Alt Gr` + key for common keys
- Use Compose Keys
